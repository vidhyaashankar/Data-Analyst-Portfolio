Diwali Sales Data Analysis Using Python

Introduction

This project involved analyzing a retail sales dataset during the Diwali festival season. The goal was to identify customer buying behavior, analyze sales trends, and extract key insights based on demographics, product categories, and spending patterns. The analysis was performed using Python and core data analytics libraries.

Objective

To perform exploratory data analysis (EDA) using Python to understand Diwali sales performance, customer segments, and revenue drivers, and deliver actionable insights for marketing and product planning.

Methodology

Data Loading & Initial Exploration:

  1. Loaded the dataset using pandas.read_csv().
  2. Reviewed the shape, structure, column names, and data types using .head(), .info(), and .describe().

Data Cleaning:

  • Handled missing values using .dropna() and .fillna().
  • Converted Amount column from float to integer using astype(int).
  • Renamed Marital_Status values to either Married or Unmarried for simplification using apply().

    • Feature Selection:

  • Dropped irrelevant columns like Status, unnamed, and duplicates.
  • Selected key features such as Gender, Age, State, Product_Category, and Amount for analysis.

    • Exploratory Data Analysis (EDA):

  • Performed univariate and bivariate analysis using:
    • value_counts(), groupby(), agg()
    • Visualized with Matplotlib and Seaborn:
      1. Bar plots: Gender-wise and age-wise spending.
      2. Count plots: Category-wise purchasing behavior.

    Identified:

    • Top states by total orders and revenue.
    • Gender and age groups with highest spending.
    • Most purchased product categories.

    Insight Generation:

  • Found that married women aged 26â€“35 in major metro states contributed the highest revenue.
  • Identified top-selling product categories and potential marketing targets.
  • Provided recommendations for promotional campaigns and inventory focus based on customer insights

    • Results

    Delivered a complete exploratory data analysis (EDA) report with visualizations.

    Highlighted:

    • High-value customer segments based on gender, age, and marital status.
    • States and cities with the highest demand.
    • Product categories with the most revenue generation.
